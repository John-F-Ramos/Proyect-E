#🎓 CEUTEC Equivalencias
Sistema web para la gestión de equivalencias académicas y seguimiento del progreso curricular de estudiantes de CEUTEC.

Plataforma que permite a estudiantes importar su historial académico (desde Excel o texto plano), visualizar su progreso en la carrera, gestionar equivalencias automáticas (especialmente para estudiantes bilingües) y simular cambios de carrera con proyección de materias equivalentes.

#Tecnologías
Backend
Node.js + Express - Servidor y API REST
SQL Server - Base de datos relacional
express-session - Manejo de sesiones
bcryptjs - Encriptación de contraseñas
multer - Subida de archivos
exceljs - Procesamiento de archivos Excel

Frontend
EJS - Motor de plantillas
TailwindCSS - Estilos y diseño responsive
Chart.js - Gráficos interactivos
JavaScript Vanilla - Lógica del cliente

#Características Principales
Importación inteligente: Carga de historial desde Excel o texto plano con detección automática de columnas
Dashboard personalizado: Progreso curricular, rendimiento académico y materias en curso
Sistema de bilingües: Equivalencias automáticas para estudiantes con ING113
Simulador de carrera: Proyección de materias equivalentes al cambiar de carrera
Catálogos académicos: Gestión de carreras, planes de estudio y materias
UPSERT inteligente: Actualización de registros sin duplicados
